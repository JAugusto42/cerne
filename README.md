# Cerne

A tool based in TUI (Terminal User Interface) for SCA analyses.

## Functionalities
- Tree dependency visualizer.
- Vulnerability detection via OSV database (osv.dev).
- Support: Go, package-lock.json and Gemfile.lock

## Installation
```bash
pip install cerne
```

## Usage
Run cerne on app directory or using cerne --path /path/to/app
